getusername

PURPOSE

GETUSERNAME Get user name.

SYNOPSIS

function userName = getusername

DESCRIPTION 

 GETUSERNAME  Get user name.
       USERNAME = GETUSERNAME returns the login name of the current MATLAB
       user. Function should work for both Linux and Windows.
