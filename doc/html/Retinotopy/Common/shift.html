shift

function res = shift(mtx, offset)

DESCRIPTION 

 Shifts 2D matrix circularly.
 [RES] = shift(MTX, OFFSET)
 
 Circular shift 2D matrix samples by OFFSET (a [Y,X] 2-vector),
